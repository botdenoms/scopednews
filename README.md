# Weeklynews
a scoped filter based news scrapper from your sources list of new sources

Developed in python programming language using [Beautifulsoap](), [request]()

## Usage 
sources.py

    used to manage your sources list, add new sources, remove & clean the file.
scope.py

    used to manage your scopes list, add new scope, remove & clean the scopes used to filter your the new being tracked.

main.py

    used to run the main process of fetching the sources, extracting the news, filtering the news based on the scope list words & return the succesful hits.

## Constrains
Extracted news are limited/scoped to the words listed in the scopes list file

The new sources are limited to the sources list, add more source to diversify your news more different point of views.
